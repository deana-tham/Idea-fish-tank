# Product Requirements Document

## Product Name

Idea Fish Tank

## Overview

Idea Fish Tank is a playful productivity tool that helps users capture and revisit their ideas in a visual and engaging way. Each idea is represented as a fish swimming inside a digital aquarium. When an idea is completed, the fish becomes shiny to symbolize achievement.

## Problem Statement

People frequently have ideas throughout the day but do not have a consistent place to capture them. Ideas are often scattered across notebooks, sticky notes, or multiple apps. This fragmentation leads to forgotten or lost ideas.

## Goal

Create a fun, centralized place where users can store and visualize their ideas.

The product should:

- make idea capture fast and easy
- encourage revisiting old ideas
- make idea storage feel engaging rather than like a boring list

## Target Users

Primary users (besides me):

- creative thinkers
- students
- product builders
- entrepreneurs
- people who frequently brainstorm ideas

## User Stories

User Story 1

As a user, I want to quickly add an idea so I don’t forget it.

User Story 2

As a user, I want to see my ideas visually represented so they feel alive and memorable.

User Story 3

As a user, I want to mark ideas as completed when I execute them.

User Story 4

As a user, I want to revisit old ideas easily.

## Core Features

Idea Input

Users can type and save ideas quickly.

Idea Visualization

Each idea becomes a fish inside the tank.

Fish Animation

Fish float around the screen slowly to create a relaxing visual environment.

Idea Details

Clicking a fish opens the idea details.

Idea Completion

Users can mark an idea as completed.

Completed ideas become shiny fish.

## MVP Scope

For the first version we will include:

- idea input field
- idea storage
- fish generation for each idea
- fish animation
- idea completion toggle
- shiny fish for completed ideas

## Success Metrics

- number of ideas added
- number of completed ideas


## Future Opportunities

- multiple tanks
- idea categories
- idea reminders
- gamification elements
